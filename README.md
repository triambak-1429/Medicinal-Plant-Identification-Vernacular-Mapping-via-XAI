Medicinal Plant Identification & Vernacular Mapping via XAI | Python, EfficientNetV2, SHAP, LIME, Grad-CAM

Developed a lightweight computer vision classification model using EfficientNetV2 to identify structurally similar South Indian medicinal plants.

Engineered a vernacular mapping pipeline to translate model outputs from Latin scientific names to local languages, bridging the gap between traditional ethnobotanical knowledge and AI accessibility.

Validated the deep learning "black box" model by integrating Explainable AI (XAI) frameworks including Grad-CAM, LIME, and SHAP to ensure the model was classifying based on actual leaf structures rather than background artifacts.

Authored a comprehensive research paper detailing the architectural efficiency and the ethnobotanical applications of the model.
